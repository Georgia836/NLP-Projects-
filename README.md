# NLP Exercises

This repository contains a collection of Natural Language Processing (NLP) exercises implemented in Python. The projects cover fundamental techniques in text processing, statistical language modeling, word embeddings, and neural networks for sentiment analysis.

The goal of this repository is to demonstrate practical understanding of core NLP concepts through hands-on implementation using libraries such as NLTK, TensorFlow, and scikit-learn.

## Projects

### Exercise 1 – Regular Expressions and Byte-Pair Encoding
Introduction to NLP text processing techniques using Python and NLTK. Includes information extraction with regular expressions on the Gutenberg corpus and an implementation of the Byte-Pair Encoding (BPE) algorithm for subword tokenization.

### Exercise 2 – N-gram Language Models and Word Embeddings
Implementation of trigram language models with Laplace smoothing and evaluation using perplexity. The project also explores semantic similarity using FastText word embeddings and performs sentiment classification on the IMDb dataset using Logistic Regression.

### Exercise 3 – Neural Network for Sentiment Analysis
Development of a feed-forward neural network using TensorFlow/Keras for sentiment classification on the IMDb dataset. The model includes an embedding layer, dense neural architecture, and binary classification with performance evaluation.
